GoBaAgency Web Package
----------------------
Contenido:
- index.html        (sitio principal, bilingüe ES/EN)
- landing.html      (landing optimizada para captación)
- assets/           (logo.svg, whatsapp.svg, favicon.svg)
- netlify.toml      (config para Netlify)

Instrucciones rápidas:
1) Descomprime el paquete.
2) Sube los archivos a Netlify o Vercel:
   - Netlify: arrastra la carpeta al área de 'Drag and drop your site output folder'
   - Vercel: usa 'New Project' -> 'Import' -> 'Other' y sube los archivos
3) En Netlify/Vercel configura tu dominio 'GoBaAgency.com' y verifica DNS.
4) Los formularios envían a: yansr20@yahoo.com
5) WhatsApp: +54 9 11 6572 8549
